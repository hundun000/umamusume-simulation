## 开发中

游戏 《赛马娘 Pretty Derby》 比赛模拟。

公式和数值参考原作游戏的解包研究。以模拟的游戏帧进行一场比赛，即模拟赛马每一帧时的位移量，速度，加速度。

模拟结果的记录形式和输出形式可选择多种实现。
